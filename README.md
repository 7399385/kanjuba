# 看剧吧
![](https://raw.githubusercontent.com/Sohnia/kanjuba/master/app/src/main/res/mipmap-xxxhdpi/ic_launcher.png)

用Jsoup爬取网上的视频资源库。
加入第三方接口进行整合解析，使用腾讯信鸽进行接口更新，
X5内核有点bug但是可以使用，后期会修改。
